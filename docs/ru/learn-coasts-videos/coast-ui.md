Coastguard — это веб-интерфейс, который вы получаете с `coast ui`. В этом видео показан обзор интерфейса и информации о среде выполнения, которую он предоставляет — состояние экземпляра, сопоставления портов, журналы и состояние сервисов с первого взгляда.

Полное справочное описание см. в [Coastguard](../concepts_and_terminology/COASTGUARD.md).
