# Coasts Video Course

## Lessons

1. [Coasts](coasts.md) — what a Coast is and how the core model works.
2. [Ports](ports.md) — how Coasts handle port isolation and parallel runtime access.
3. [Assign](assign.md) — switching a running Coast between worktrees.
4. [Checkout](checkout.md) — bringing a Coast onto your canonical ports for active use.
5. [Volumes](volumes.md) — how Coasts deal with volumes and persistent service state.
6. [Secrets](secrets.md) — managing secrets inside a Coast.
7. [Getting Started](getting-started.md) — a hands-on walkthrough for trying Coasts on a real project.
8. [Coast UI](coast-ui.md) — the Coastguard UI and the runtime information it exposes.
